# 権力支持基盤理論
支持基盤理論

　政治家はなぜ国民のために働かないのか、あなたはこう思ったことがあるのでしょうか。もちろんそのような目標を持つ政治家が少ないわけではないでしょう。しかし、実際に出てくる政策の中には大多数の国民に不利益のものがあったり、多くの国民に嫌われているように見えても何故か毎回当選する政治家がいたりします。「朕は国家なり」という言葉があります。17世紀のフランス国王ルイ14世による絶対王政を象徴する言葉です。この言葉だけを聞くと、ルイ14世は万能で、何でも自分の思うままに国を動かせたと思いがちですが、そうではありません。どんなに優秀な独裁者であっても、１人だけで国を運営することは決してできません。例えば国を敵から守るには軍隊、国内の治安を守るには警察、税金を徴収したり、道路を作るなど何をするにも誰かほかの人を頼らなければなりません。つまり、独裁者は決して独り（ひとり）ではないのです。独裁者の周りには必ず仲間がいて、その仲間を如何に自分の思い通りに動かせるかが独裁者として成功するかどうかのカギになります。独裁者が集中するべきなのは如何に周りの人間を自分に従わせるか、どうすれば自分のために人々が働いてくれるのかを考える。これこそが独裁者に与えられた仕事なのです。
 
 　権力を握る上でまず知るべきなのは国民には三種類の人間がいることです。それが「名目有権者」・「実質有権者」・「盟友」です。まず名目有権者は文字通り名目上指導者を選ぶ力を持つ人。実質有権者は実質上指導者を選ぶ力を持つ人を指します。例えば日本では18際以上の全ての国民が原則として選挙権を持つので、子供以外はほぼ全員が名目有権者になります。しかしその中で実際に選挙権を行使するのは全員ではありません。名目有権者の中で実際に選挙に参加して選挙結果に影響を与えた全ての人を、つまり投票者が実質有権者に入ります。そして盟友とはその指導者が実際権力を握るために、最低限必要な人を言います。例えば日本の総理大臣国会議員による指名で決められ、指名されるのに最低限必要なのは、総投票数の過半数です。衆議院と参議院の議員定数の過半数はそれぞれに233人と123人なので、日本における盟友とは２つの合計である356人の議員を当選させるのに最低限必要な投票者数なのです。この数は一定ではなく、正確な数は出せませんが、例えば2017年衆議院議員選挙の比例代表に限って計算すると、まず有権者数は106,091,299人で、投票者数は56,947,831人。この内、自民党と公明党の得票数の合計が25,533,429人なので、実質有権者の45%以下、名目有権者の24%以下だったということです。以下という言葉がつく理由はこの後の指名選挙で安倍晋三氏が獲得したのは過半数を超える2/3で、過半数を取るだけならば、さらに少なくても可能なので、実際の盟友はさらに少なかったと言えます。ちなみに北朝鮮でも全ての成人に選挙権が与えられているので、名目有権者の多さは他の国と大差ありません。そして全国民が強制的に投票させられるので、投票率が毎回100%になります。ただし、そのほとんどは実質有権者とは言えません。なぜなら候補者は事前に共＊党が用意した者だけなので、ほんとの実質有権者とは候補者を指定する共＊党の上層部、およそ数千人で、盟友つまり金正恩総書記が上に立つのに必要とするのはほんの200人ほど、全人口のおよそ0.0008%と言われています。北朝鮮のような変わった例もありますが、ここで重要なのはどんな国にも上の三つの階級が存在するということです。これに例外はありません。中国のような民主国家でも、北朝鮮のような独裁国家であっても、この構造は必ず存在しており、三つの大きさは国によってバラバラです。ただ独裁国家には特徴があります。それは盟友が少なく、名目有権者が多いという点です。一方で民主国家で名目有権者が多いのも同じですが、違うのは盟友も同じく多い点です。ではなぜ北朝鮮の盟友はこれほど少ないのでしょうか。
  
  実はこれが独裁をする上でカギとなる第一の原則「盟友を少なくする」になります。盟友の定義をもう一度確認してみます。盟友とは指導者が権力を握る上で最低限必要な人のことです。つまりこれが何を意味するかと言うと、「盟友の支持さえあれば権力が握れる」ことです。では具体的にどうすれば盟友の支持を得られるのでしょうか。ここで原則一に続く原則二「盟友に十分な報酬を与える」です。軍隊を自分に従わせたいならば、軍隊に報酬を、警察を従わせたいのなら報酬を。他にも必要な人間全員を盟友として報酬を与えます。報酬はお金に限らず、例えば物品でも良いですし、特権を与えたり、道路や病院を近くで作ってあげても構いません。こうすると盟友たちは報酬を目当てに指導者を支持するようになります。独裁国家と民主国家では一人一人の盟友が受け取る報酬に違いが出ます。まず独裁国家の盟友は報酬を多く受け取ります。なぜなら国民に対して盟友の数が少ないので、報酬を狭く厚く分配できるからです。基本的に独裁国家のほうが経済格差が大きいのがこのせいで、報酬が少数の盟友に偏る一方で、大多数の国民はほぼ何も受け取らない結果、少数の盟友が大金持ちに、逆に国民の多くは貧乏になるのです。盟友が住む地域には道路が隅々まで敷かれて、一流の学校や病院が整備されます。反対に他の国民の方は十分な教育や医療は受けられないので、教育水準が下がって不健康なり、国全体が発展しなくなります。北朝鮮の共＊党幹部が高級住宅街で高級車を乗り回している間に、庶民は飢えに苦しんでいるのがまさにその典型例です。一方民主国家の方は逆で、報酬が広く薄く分配されます。なぜなら盟友が多い分どうしても一人あたりの報酬を少なくせざるを得ないからです。盟友の数が多いと、報酬の配り方も工夫する必要が出てきます。盟友の数が少なければ、一人一人の好みを聞いて、好きなものを与える余裕がありました。しかし盟友が多すぎる場合、一人一人にほしいものを聞いて回るのは不可能なので、解決策として、どの盟友も欲しがるような普遍的なもの、例えば道路や消防、水道などの公共財が報酬として与えられます。本来この報酬は盟友のためだけのものですが、公共財を配る場合、盟友以外もその恩恵を享受できます。なぜなら盟友が多いと北朝鮮のように盟友を一か所に集めることができないので、バラバラの場所に住む、よって道路も全国に満遍なく広げる必要もあります。これはあくまで盟友のための道路ですが、盟友以外の人に道路を使用禁止にすることは事実上不可能なので、盟友以外にも恩恵がもたらされます。このような仕組みがあり、民主国家では比較的経済格差が小さくなります。あなたが総理大臣を支持するしないに関係なく、道路や水道が使えるのはひとえに盟友が多いからなのです。
  
  独裁国家は狭く厚く、民主国家は広く薄く。この報酬分配の仕組みは全て盟友の数次第というのが本質です。民主国家では多くの盟友に支持してもらわなければならないので、権力維持の難易度もその分高く、民主国家の指導者が比較的頻繁に交代するのもこれが理由です。反対に独裁国家の指導者は盟友が少なく、報酬の調整もやりやすいので、権力の安定に向いていると言えます。しかし、ここで忘れてはいけないのは、ただ盟友を少なくするだけでは問題があるということです。あまりにも少ない人に依存してしまうと、それだけ一人あたりの発言力も増してしまうのです。「支持して欲しいなら、もっと報酬を出せ」と際限なく求められたり、しまいには自分の権力自体を奪おうとするかもしれません。つまり盟友が自分に従う程度の報酬はあげても、依存しすぎてはいけないということです。ではどうすればこの問題を解決できるでしょうか。
  
  ここで必要な新たな心得が原則三「名目有権者を多くする」です。名目有権者は盟友の候補者と考えましょう。盟友の後ろに多くの名目有権者を置いて、盟友が少しでも反抗する素振りを見せれば、すぐに名目有権者と交換するのです。
